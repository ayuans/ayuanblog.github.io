炸鸡翅鸡排的正确做法！

准备食材with注意事项

腌制食材

  • 牛奶 350ml
  • 柠檬 1个
  • 洋葱圈半个
  • 大蒜 1个
  • 盐 15g
  • 黑胡椒 5g
  • 辣椒粉 5g
  • 辣椒面 3大勺
  • 鸡蛋 2个

    步骤：

  1. 将柠檬洗干净，将整个柠檬汁加入牛奶，静置5分钟。
  2. 成为白脱牛奶之后，将以上食材放入白脱牛奶，搅拌均匀。
  3. 放入鸡腿。
  4. 用保鲜膜密封后放入冷藏，腌制8小时，最好隔夜。

面壳食材

  • 面粉 300g
  • 淀粉 80g
  • 辣椒粉 1大勺
  • 黑胡椒 1小勺
  • 盐 1小勺
  • 炮打粉 1小勺

    步骤：

  1. 将腌制好的鸡腿放入面粉中翻滚！
  2. 油温160°C，炸5分钟。
  3. 时间到了，取出放置一段时间。
  4. 油温180°C,炸30秒。
